# 🎮 Respawn Diário

**O ponto de renascimento diário de todo gamer que vive, joga e respira videogames.**

Respawn Diário é mais do que um blog — é o seu checkpoint gamer diário, onde cada post é uma nova oportunidade de descobrir conteúdo de qualidade, análises honestas e discussões relevantes sobre o universo dos videogames.

## ✨ Características

### 🎨 Design Moderno
- Interface dark mode otimizada para gamers
- Design responsivo que funciona em todos os dispositivos
- Animações suaves e efeitos visuais atraentes
- Tema gaming com cores neon e gradientes futuristas

### 📱 Totalmente Responsivo
- Mobile-first design
- Otimizado para smartphones, tablets e desktops
- Navigation drawer em dispositivos móveis
- Grid system flexível

### 🚀 Performance
- CSS otimizado com variáveis customizadas
- JavaScript modular e eficiente
- Lazy loading para imagens
- Animações GPU-aceleradas

#### 👥 Sobre a Equipe

#### 📞 Contato

#### 🎮 Recursos Especiais

## 🛠️ Tecnologias Utilizadas

### Frontend
- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Design moderno com:
  - CSS Grid e Flexbox
  - Custom Properties (variáveis CSS)
  - Animations e Transitions
  - Media Queries para responsividade
- **JavaScript ES6+** - Interatividade com:
  - DOM manipulation
  - Intersection Observer API
  - Local Storage
  - Modular architecture

### Recursos Externos
- **Google Fonts** - Typography (Orbitron + Inter)
- **Font Awesome** - Ícones vetoriais
- **Unsplash** - Imagens de placeholder

## 🚀 Como Usar

### 1. Clone ou Download
```bash
git clone https://github.com/seuusuario/respawn-diario.git
cd respawn-diario
```

### 2. Abra no VS Code
```bash
code .
```

### 3. Use Live Server
- Instale a extensão "Live Server" no VS Code
- Clique com botão direito no `index.html`
- Selecione "Open with Live Server"

### 4. Ou abra diretamente
Simplesmente abra o arquivo `index.html` no seu navegador favorito.
